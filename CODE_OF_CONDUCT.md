# Code of Conduct

Under-Utils 希望维护一个专业、开放、尊重事实的协作环境。所有参与者都应遵守以下准则。

## Expected Behavior

- 聚焦技术问题本身，给出可验证的论据。
- 尊重不同经验背景的贡献者。
- 在 review 中指出风险、边界和可改进点，而不是评价个人。
- 接受维护者基于项目定位、维护成本和长期一致性做出的取舍。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视或威胁。
- 在公开渠道泄露他人隐私、密钥、内部系统信息或安全漏洞细节。
- 反复发布与项目无关的内容、广告或低质量噪声。
- 明知不符合项目定位仍持续推动无关争论。

## Enforcement

维护者可以根据情况采取提醒、编辑或删除内容、限制参与、关闭 issue/PR 等措施。严重违反准则的行为可能导致永久禁止参与项目。

如果你发现违反准则的行为，请通过 issue 或维护者提供的私有渠道报告，并尽量附上上下文链接和必要事实。
